# Codex Conventions

- Usa ids canonicos en kebab-case para skills en configuracion.
- Aplica cambios minimos y trazables en `.codex/**` y `AGENTS.md`.
- Ejecuta validadores de `.codex/tools/` antes de cerrar cambios de configuracion.
- Usa `.codex/rules/README.min.md` para cambios de configuracion Codex.
- Para estandares de desarrollo del proyecto, usa `./.agents/skills/vass-config/README.min.md`.
